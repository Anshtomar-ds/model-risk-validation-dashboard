# Credit Risk Model Validation Dashboard

This project demonstrates end-to-end model validation and monitoring for a Credit Risk Logistic Regression model.

## Features

- Model Training
- Model Validation
- AUC, Gini, KS Statistic
- Population Stability Index (PSI)
- Model Monitoring Dashboard
- Streamlit Application

## Tech Stack

- Python
- Pandas
- Scikit-Learn
- Streamlit
- Matplotlib
